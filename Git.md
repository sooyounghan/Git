### CLI / GUI
-----
1. CLI(Command Line Interface) : 명령어 기반 인터페이스 (Git Bash)
   - 뭔가를 실행하기 위해 어떤 명령을 내릴 때 사용하는 것이 좋음
2. GUI(Graphic User Interface) : 유저가 사용하기 편하도록 그래픽 요소를 활용한 인터페이스 (Source Tree)
   - 프로젝트의 상태를 Git 상에서 자세히 살펴보고 싶을 때 사용하는 것이 좋음
